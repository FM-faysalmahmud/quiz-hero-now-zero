Q-2: What are the differences between local storage, session storage and cookies?

Ans: The difference between them are local storage stores the data in your browser and if you exit the browser and only the browser again then the data still remains in the browser for a certain period of time. But in session storage you can store the data but it only stays on the browser as long as the browser is on. If you exit the browser then the data will also be gone. Cookies are also similar, it also stores data. When you are browsing a website then the site stores some cookie data on your browser.
